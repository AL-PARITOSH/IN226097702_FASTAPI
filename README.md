# FastAPI Practice Repository

This repository contains practice work and experiments built using **FastAPI**, a modern and high-performance web framework for building APIs with Python.

The purpose of this repository is to practice and understand the fundamentals of **backend API development**, including creating endpoints, handling parameters, filtering data, and testing APIs using FastAPI’s built-in documentation tools.

This repository can be reused for different FastAPI assignments and exercises.

- Python 3

- Uvicorn

## Running the FastAPI Application

### 1. Install dependencies

pip install fastapi uvicorn

### 2. Run the FastAPI server

uvicorn main:app --reload

### 3. Open the application in the browser

http://127.0.0.1:8000

---

## API Documentation

FastAPI automatically provides interactive documentation using **Swagger UI**.

Open the following link:

http://127.0.0.1:8000/docs

Swagger UI allows you to:

- View available API endpoints
- Test APIs directly in the browser
- See request and response formats
- Understand API parameters
